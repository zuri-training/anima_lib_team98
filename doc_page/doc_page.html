What is Animax?

Animax is a simple animation librarythat allows users perform web animation writing little to no code to achieve that.

Getting Started

Animating your project with self written code is quite stressful and time consuming but not to worry, ANIMAX offers you world class tool that stands up to the most rigorous implementation

Features

Normal Language: The names of the animation are very close to normal human language and users do not need to write any code at all.

Easy to Use: The library navigation is clear and flows. It gives you control on how you want the animation to look like

Light and user-friendly: It is built on pure CSS and doesnt take much load time hence content loads a lot faster.

Comparison with Other Libraries

Across multiple Animation libraries, Animax has a unique forcus on animation libraries and how it can be easily used by developers.

We have checked out other animation libraries and would like to offer insights to help you hopefully navigate through the choices you have.

GSAP

GreenSock is an Opensouce tool that works around countless browser inconsistencies and features the ScrollTrigger, SplitText, ScrollSmoother and ScrollMagic animation

With their strength lying in timeline classes for synchronising animations, ability to stop reverse or yoyo animation, animating along Bezier curves, observer for normalizing event detection across browsers.

One thing which Animax does well is correct syncing of the keyframes regardless of the value entered. Some of the features of Animax was inspired by the best things about GSAP and itâ€™s a great alternative.

Anime.js

Anime.js is another popular animation library that works with CSS properties, SVG, DOM attributes and Javascript objects featuring layered CSS transform, controls and callbacks

It has built-in straggering system that makes complex follow through and overlapping animations. So if you need simple overlapping animations, you can check it out.

Animate.css

Animate.css is another very popular animation libraries which is great for emphasis, homepages and sliders. It features cross-browser animations, bounce and zoom effect, copy & paste.

It has many similarities with Animax- both being able to set delay and speed and also bounce effects. However there is no feasible way to troubleshoot problems users might have.

Something Missing?

If you find issues with the library or have suggestions on how to improve the library or the project in general, please file an issue for us or fill in the feedback form.

If you have an idea for a new feature, please raise an issue to discuss it before sending code. In many cases features are absent for a reason.
