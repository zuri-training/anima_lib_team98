Usage Examples

Simple examples for animating your components.

Animating a Component

              
                <script type="text/javascript" src="script.js"/script>

Bounce

A component that can be made to bounce like a ball, it has the options to bounce or slide and bounce.

Blink

Animax provides a blink animation that can be modified to be blink and fade, blink red, blink blue and blink green.

Fade

Fade allows elements to blend out of view with the options to Fade In, Fade Out, FadeInOut and Persist.

Alternate Colours

Altenate colour changes the colour of the object with the option to alternate an object colour or a text colour.

Stretch

An object or component is stretched to a particular length as it is in the viewport with option to stretch wide, stretch high or squash and stretch.

Cloud

The objects becomes more blurry and the only option available is big cloud.

Pulse

The object is given a pulsating effect that changes its shape and opacity. The heartbeat and pulse in is available.
